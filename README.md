## TypeScript

Es un supraconjunto de JavaScript, que quiere decir esto, que posee todas las caracteristicas de de JavaScript pero ofrece otras más. TypeScript puede consumir JavaScript.

Aún asi, no todo el código TypeScript es JavaScript, ya que implementa algunas nuevas caracteristicas, cómo los tipos estáticos, permite que sea más sencillo el desarrollo y menos propenso a errores.
